# Late Antiquity

Late Antiquity marks a period that reshaped institutions, argument styles, and dominant explanatory models. In this corpus it is tied to [[aristotle]] and the development of [[forms]].

## Context

This period is best studied by combining textual analysis with place-based history: see [[rome]], [[athens]], and [[konigsberg]].

- Representative figure: [[aristotle]]
- Representative idea: [[forms]]
- Comparative references: [[classical-greece]], [[enlightenment]], [[virtue-ethics]]
