# Islamic Golden Age

Islamic Golden Age marks a period that reshaped institutions, argument styles, and dominant explanatory models. In this corpus it is tied to [[avicenna]] and the development of [[rationalism]].

## Context

This period is best studied by combining textual analysis with place-based history: see [[baghdad]], [[athens]], and [[konigsberg]].

- Representative figure: [[avicenna]]
- Representative idea: [[rationalism]]
