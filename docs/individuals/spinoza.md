# Baruch Spinoza

**Lifespan**: 1632 - 1677

**Nationality**: Dutch

## Ideas (1)

### [[ideas/substance-monism|Substance Monism]]

**Year**: 1677

**Period**: [[periods/early-modern-philosophy|Early Modern Philosophy]]

The doctrine that there is only one substance in the universe, which Spinoza identified with both God and Nature (Deus sive Natura)
