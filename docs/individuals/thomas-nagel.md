## Ideas (1)

### Subjective Experience and "What Is It Like?"

**Year**: 1974

**Period**: [[periods/contemporary-philosophy|Contemporary Philosophy]]

Consciousness has subjective, experiential aspects that cannot be captured by objective, physical description - "What is it like to be a bat?"

**Tags**: #Philosophy of Mind, #Consciousness, #Subjectivity

**Influenced by**:
- critiques: Physicalist reductionism
