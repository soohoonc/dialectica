# Is-Ought Problem

**Author**: [[philosophers/david-hume|David Hume]]

**Year**: 1739

**Historical Period**: [[periods/enlightenment|Enlightenment]]

**Tags**: #Epistemology, #Ethics

## Description

The difficulty of deriving normative conclusions from purely factual premises
