# Code of Hammurabi

The Code of Hammurabi is a Babylonian legal text composed c. 1754 BCE during the reign of King Hammurabi of Babylon.

## Description

The code is inscribed on a black diorite stele standing over 7 feet tall. At the top, Hammurabi is depicted receiving the laws from Shamash, the god of justice.

## Content

The code consists of 282 laws covering:

- Property rights
- Trade and commerce
- Family relations
- Labor disputes
- Criminal punishments

## Significance

One of the oldest deciphered writings of significant length, the Code of Hammurabi provides invaluable insight into Mesopotamian society and influenced later legal codes.

## Famous Principle

"An eye for an eye, a tooth for a tooth" - the principle of proportional justice (lex talionis).
